# LectureLens End-User Operations Guide 📖

## 🔌 Hardware Setup Architecture
- Minimum Hardware Context: 8GB RAM / Core i5 CPU or Apple Silicon M-Series.
- External Application Dependencies: Ensure `Ollama` is executing locally in background processing layers.

## 🚀 Step-by-Step Processing Tasks
1. Execute engine framework invocation command: `streamlit run app/app.py`
2. Drop long lectures (`.mp3`, `.wav`) or technical lecture PDF data logs directly into the file uploader layout window.
3. Track active text chunk caching pipelines through real-time state visualization percentage dashboards.
4. Export finalized flashcard blocks as clean local markdown data dumps.
